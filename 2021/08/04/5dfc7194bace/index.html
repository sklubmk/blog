[프로그래머스] (LV1) 평균 구하기

평균 구하기

JavaScript, Python


문제 설명

정수를 담고 있는 배열 arr의 평균값을 return하는 함수, solution을 완성해보세요.

제한사항

  • arr은 길이 1 이상, 100 이하인 배열입니다.
  • arr의 원소는 -10,000 이상 10,000 이하인 정수입니다.

입출력 예

arr return
[1,2,3,4] 2.5
[5,5] 5


풀이

javascript


python


설명

기본 문법 구현능력을 묻는 문제.
